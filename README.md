# TRSS- Tresnox Race Simulation System 
#### ℹ️ A GUI program that simulates F1 in Schools races, and provides you with the results

#### TRSS provides you with:
* *Predicted race times*
* *Displacement from origin every 50ms*
* *Distance travelled every 50ms*
* *Velocity every 50ms*
* *Distance-Time and Velocty-Time graphs*
* *Point of deceleration*
* *Max velocity achieved*
* *Final velocity achieved*
